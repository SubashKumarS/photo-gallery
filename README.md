# 📸 Photo Gallery

A dynamic **Photo Gallery Web Application** that allows users to generate and view random images using the **Unsplash API**.

Users can enter a number between **1 and 10**, and the application will fetch that many random photos and display them in a responsive gallery layout.

---

## 🚀 Features

* 🔢 Enter number of images to display (1–10)
* 📷 Fetch random photos from the **Unsplash API**
* ⚡ Dynamic image loading using **JavaScript**
* 🎨 Clean and responsive gallery layout
* ❗ Input validation with error message
* ⏳ Loading spinner while fetching images

---

## 🛠️ Technologies Used

* **HTML** – Structure of the web page
* **CSS** – Styling and responsive layout
* **JavaScript** – API requests and dynamic image rendering
* **Unsplash API** – Fetch random images
